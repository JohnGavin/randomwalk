### About This Simulation

This dashboard provides an interactive interface for the `randomwalk` R package.

- **Author**: John Gavin
- **Source Code**: [https://github.com/JohnGavin/randomwalk](https://github.com/JohnGavin/randomwalk)

This simulation is a simple random walk that builds a fractal graph. It is NOT a DLA (Diffusion-Limited Aggregation) simulation.
