# randomwalk: Asynchronous Pixel Walking Simulation

Implements parallel random walk simulations that create fractal graphs
through asynchronous pixel walking on a grid. Features include true
asynchronous parallel processing, real-time grid state synchronization,
comprehensive statistics tracking, and an optional Shiny interface.

## Main Functions

- [`run_simulation`](https://johngavin.github.io/randomwalk/reference/run_simulation.md):

  Run a complete simulation

- [`initialize_grid`](https://johngavin.github.io/randomwalk/reference/initialize_grid.md):

  Create a new grid

- [`create_walker`](https://johngavin.github.io/randomwalk/reference/create_walker.md):

  Create a walker object
